# 🌤️ AI Weather Telegram Bot — n8n Automation

An intelligent daily weather report bot built with n8n that sends AI-generated weather updates to Telegram every morning.

---

## 🚀 What It Does

- ⏰ Triggers automatically every day at **7:00 AM**
- 🌍 Fetches **live weather data** for Karnal, Haryana via OpenWeatherMap
- 🤖 Passes data to **Google Gemini AI** which generates a premium, human-friendly weather report
- 📲 Sends the final message directly to your **Telegram** chat

---

## 🧠 Tech Stack

| Tool | Purpose |
|------|---------|
| n8n | Workflow automation |
| OpenWeatherMap API | Live weather data |
| Google Gemini (PaLM) | AI message generation |
| Telegram Bot API | Message delivery |
